# Kavy React Portfolio

A responsive React/Vite portfolio inspired by the reference screenshot.

## Run locally

1. Install Node.js LTS.
2. Open this folder in VS Code.
3. Open the terminal and run:

```bash
npm install
npm run dev
```

4. Open the local URL shown in the terminal.

## Customize

Edit these files:

- `src/App.jsx`: name, email, education, links, projects and text.
- `src/styles.css`: colors, spacing and design.
- `public/kavy-profile.jpg`: replace the profile photograph while keeping the same filename.

## Deploy to Vercel

1. Upload the project to a GitHub repository.
2. Sign in to Vercel using GitHub.
3. Click **Add New → Project**.
4. Import your repository.
5. Click **Deploy**.

Vercel should automatically detect Vite.
